# Academic Writing Style — 学术写作规约

任何由 `write-*` 系列 skill 产出的论文段落、由 `manage-bibliography` 编辑的引用、由 `render-paper` 拼装的最终 PDF，必须遵守本规约。**违反任一条 = 内容不可对外发表**。

## 一、禁忌（Hard "No"）

下列内容**绝对不能出现**在论文正文、脚注、图表标题、参考文献条目中：

1. **机器路径** — 任何 `output/checkpoints/*.json`、`code/03_solve/*.py`、`results/figures/*.pdf` 等仓库内路径。论文读者不需要知道你的目录结构。
2. **流水线 / 实现术语** — "pipeline"、"by the pipeline"、"automated workflow"、"sub-agent"、"`math-reviewer`"、"`verifier`"、"plugin"、"AI-assisted"、"Codex"、"Claude" 等任何**暴露这是由 agent / 流水线生成的**词语。论文应该读起来像是研究者亲手写的。
3. **命令行片段** — 任何 `python ...`、`bash ...`、`pdflatex ...`、`git ...` 等 shell 命令。这些属于 README / 复现说明，**不属于**论文正文。
4. **元元注释** — "this paper is generated by"、"see also the corresponding JSON file"、"as logged by the quality-report"。
5. **"Reproducibility" 章节里放 bash** — 复现声明可以有，但用一句话表述（"All code and data are available at the project repository"），**不**贴命令。
6. **过满表述** — "All variables return to steady state at the horizon"（实际未必）；"The model fits the data perfectly"；"Our results show conclusively"。一律改为可被支撑的弱声明（"approximately return to steady state"、"matches X moments closely"、"results suggest"）。

## 二、必备（Hard "Yes"）

1. **完整学术骨架**。一篇标准经济学论文必须包含：
   - **Title + Authors + Affiliations**
   - **Abstract** （150–250 字，无实现术语）
   - **1. Introduction** （motivation / literature review / contribution / roadmap 四段法）
   - **2. Model**（environment → agents → equilibrium definition → FOC）
   - **3. Calibration / Data**（参数表 + 来源；若有微观数据则附数据描述）
   - **4. Solution Method**（线性化策略、稳定性条件证明 / 数值验证）
   - **5. Results**（核心数字 + **经济学解读**，不是光列峰值）
   - **6. Robustness / Sensitivity**（至少 1–2 项稳健性）
   - **7. Conclusion**
   - **References** （BibTeX，按 AEA 引用规范）
   - **Appendix**（可选：proofs、additional figures、data appendix）
2. **Introduction 四段法**（每段一段，不交叉）：
   - § motivation：问题为什么重要 + 现实背景
   - § literature：与最近 5–10 篇文献的关系，明确点名
   - § contribution：本文新增的 1–3 条具体贡献
   - § roadmap："Section 2 ...; Section 3 ..."
3. **Results 必有经济学解读**。仅给数字不够；要解释 *为什么* 这个数字出现、与机制叙述如何对应、与已有文献相比是大是小。
4. **图表自包含**（self-contained captions）：读 caption 就能理解图，不需要回正文。
5. **正式语态**。
   - 数学定义用现在时（"The household maximises..."），不是 "will maximise"。
   - 结果叙述用现在时或过去时（"We find..." / "Output rises..."），不是 "will find"。
   - 避免口语：don't / can't / let's，写 do not / cannot / consider。
6. **引用规范（AEA / Chicago author-date）**：
   - 行内：`Smith (2020)` 或 `(Smith, 2020)` 或 `Smith and Jones (2020)`。三人及以上用 `Smith et al. (2020)`。
   - References：BibTeX `.bib` 文件，每条 entry 至少含 author, title, journal, year, volume, pages（或 publisher, address 若是 book）。

## 三、PDF 物理约束

- 字体：Latin Modern / Computer Modern（默认 `\documentclass{article}` 即可）。
- 边距：`geometry{margin=1in}`。
- 行距：默认 single（论文格式），不用 double。
- 页码：底部居中。
- 数学：用 `amsmath`、`amssymb`、`booktabs`（表格）、`graphicx`（图）。
- 图表浮动 `[t]`、`[ht]`，**不**用 `[H]`（强制位置不专业）。
- 公式编号：方程系统连续编号 `\label{eq:xxx}`，正文 `Equation~\eqref{eq:xxx}`。
- 图表交叉引用：`Figure~\ref{fig:xxx}` / `Table~\ref{tab:xxx}` 不要写 "Fig. 1"。
- 引用：`\citet{}` 用于行内（"As Smith \citet{smith2020} shows"），`\citep{}` 用于括号（"...(Smith, 2020)"）—— 即用 `natbib` 或 `biblatex`。

## 四、强制评议门

由 `write-*` skill 生成的任意段落或 `render-paper` 产出的最终 PDF，**必须**经过 `paper-reviewer` 子 agent 评审。`paper-reviewer` 检查：

- 一/二节列的所有禁忌全无
- 一/二节列的所有必备齐全
- 数字 / 引用 / 图表号能在文档内一致解析（无 `??` 警告）
- 三节列的 PDF 物理约束满足

未通过则该段 / 该 PDF **不得**进入 `paper/paper.pdf` 主文档或对外分发。

## 五、与其他规则的关系

- 与 `numerical-validation-protocol` 并行：数字必须先通过 Gate 1/2/3，**才能**被 `write-results-section` 引用进论文；本规则**不**重复审查数字本身，只审查写法。
- 与 `replication-protocol` 互补：复现声明（"All code available at..."）由本规则规定写法；技术性复现细节归 `replication-protocol` 管，但不进入论文正文。

## 六、典型反例 → 正例

| ❌ 不可写 | ✅ 改写为 |
|---|---|
| "The IRF generated by the pipeline shows..." | "Following a one-standard-deviation positive TFP innovation, output rises by..." |
| "See `output/checkpoints/rbc_ss.json` for details." | "Steady-state values are summarised in Table~\ref{tab:steady_state}." |
| "The math-reviewer sub-agent verified the dimensions." | （**删除**——这种话不上论文） |
| "Run `python solve_rbc.py` to reproduce." | "Replication materials are available at the project repository." |
| "All variables return to steady state at horizon 40." | "All variables converge toward the steady state, with capital exhibiting the slowest mean reversion (cf.\ Figure~\ref{fig:irf})." |
| "AI-generated paragraph (verifier-pending)." | （**删除评语**；段落要么进文档要么改后再进，不带状态标签） |

写作时把这个反例 → 正例表当 checklist 过一遍。
